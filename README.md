## Setup

These steps assume that you have installed Node.js already

1. From SEA_Salon, navigate to "app" directory.

    `cd src/app`

2. Install dependencies.

    `npm install`

3. Run program.

    `npm run dev`

4. Navigate to the localhost shown.


## Usage
1. See general information and services.
2. See reviews.
3. Add/update review.
4. Add Reservation(s).
